Proximity

McKay Library

Proximity helps organize the elements. Here in this example there's no doubt of which title belong to which image becasu there is a smaller space between the elements that are related to each other as if they were forming one element as a whole and a bigger space between this kind of secondary element, creating a visual difference between the icon/title to the next one.
